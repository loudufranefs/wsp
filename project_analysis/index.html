Site Intent

The primary goal of the site is to give users information about every available method to contact the client, as well as provide users with key information that they may need right away such as location and opening hours. The site also provides users with information about the services that are being offered, and information about the company and the staff.

The current website does not have an intuitive flow, I feel users will be put off by the disorganization and out-dated design, and will not be able to find information easily, not because the information isn't there, but because it may not be where users would expect it.

Information Architecture

The content in the current website, although much of the copy can be re-used, the overall content will need to be audited and re-organized. Some copy on some of the pages will be merged with more prominent pages. The main focus here is to increase the chances for users to find the information they are looking for.

User Interface

The current design is not appealing or cohesive, and does not work well for this website. A new visual design will need to be created focusing on the target audience. The new design should be friendly, clean, and give a feeling of trustworthiness or legitimacy. The layout will also be addressed, and new layout will be used to create a better overall flow of content.

Web Standards

The website uses HTML4 Transitional and does not validate with W3CValidation, The site will be rebuilt using HTML5 semantic mark-up.

Keyword Analysis:

Using Google Trends, I conducted keyword research to get an idea of any additional related keywords users in Charlotte NC were using. Keywords below are listed based on popularity from most to least popular.

  1. 1) Animal Hospital: animal medical hospital, dilworth animal hospital

The design and layout appear to be up to date. Using friendly colors and photos to give the target user a sense of comfort, and feel that this animal hospital is updated, clean and friendly. All contact information and opening hours are included in the header where users will expect to find it.
